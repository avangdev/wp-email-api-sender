This plugin reconfigures the wp_mail() function to send email using API (via AvangEmail) instead of SMTP and creates a Settings page that allows you to set up various options.

== Description ==


Avang Email Sender allows you to connect your WordPress with powerful, low-cost AvangEmail API and send unlimited emails per month!
In case of any questions or concerns, feel free to contact us anytime and report bugs

**How to get started?**
To see price and details about how to get account see https://avangemail.com

Feel free to ask question if you need help on how to configure and details about sending email
